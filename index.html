- the web
You - Internet - Servers (host files)

- html (hypertext markup language) - heart of the web
"Tags"
markup - via tags
for example: <NAME> contents </NAME> == element
1) "b" tag = bold

2) "em" tag = italics
<tag attr="value"> contents </tag>

3) "a" tag = anchor (links)
ie. <a href="www.reddit.com">derp</a>

4) "img" tag (void tag = no closing tag)
ie. <img src="url" alt="text">

 5) "br" tag (void, inline)
5a) "p" tag (not void, block) creates a box with height and width

6) "span" (inline) / "div" (block)
<span class="foo">text</span>
<div class="bar">text</div>
